# 📝 To-Do List App  

A simple command-line **To-Do List App** in Python that helps users add, view, remove, and update tasks easily.  

## 🚀 Features  
✅ Add a task to the list  
✅ View all tasks  
✅ Remove a task  
✅ Exit the application  

## 📌 How to Use  
1️⃣ Run the script:  
```bash
python main.py
```  
2️⃣ Select an option from the menu:  
- `1` ➝ Add a new task  
- `2` ➝ View all tasks  
- `3` ➝ Remove a task  
- `4` ➝ Exit
